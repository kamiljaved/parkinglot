// 	Parking-Lot Project	 //
//-------------------------------//


Note:
	Background Worker Celery is NOT required for running the project, 
	but may be used for Asynchronous performance.


Features:
	
	> Form for Adding new Vehicle
	> Notification Panel
	> Vehicle Parking List
	> Information update without page-reload


Worst Case Scenario: 
	
	Due to polling for Notifications,
	a delay of max 7.5 seconds may occur in showing a notifcation.
